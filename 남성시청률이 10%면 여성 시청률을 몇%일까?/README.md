# 🎤 K-Pop 아이돌 시청률 상관관계 분석 및 예측 프로젝트

이 프로젝트는 K-Pop 아이돌이 출연하는 프로그램의 남성 시청률과 여성 시청률 데이터를 분석하여, 두 지표 사이의 상관관계를 파악하고 머신러닝 모델을 통해 미래 시청률을 예측해보는 교육용 데이터 분석 프로젝트입니다.

## 📊 프로젝트 주요 내용

1. **데이터 전처리 (`영어 번역하기.ipynb`)**:

   - 원본 데이터(`main.csv`)의 영문 제목행을 한글로 번역하여 `제목행번역.csv`로 저장합니다.
2. **데이터 분석 및 시각화 (`main.ipynb`)**:

   - 남녀 시청률 간의 상관계수를 계산합니다.
   - 산포도(Scatter Plot)를 그려 시각적인 경향성을 파악합니다.
   - **선형회귀(Linear Regression)** 및 **다항회귀(Polynomial Regression)** 모델을 구현하여 남성 시청률이 10%일 때의 여성 시청률을 예측합니다.
3. **모델 평가 및 교육 자료 (`모델평가.ipynb`)**:

   - 결정계수(R²)와 평균제곱오차(MSE)를 활용하여 두 모델의 정확도를 비교합니다.
   - 초중고등학생을 위해 복잡한 용어를 "적중 점수"와 "틀린 거리"로 쉽게 풀어서 설명합니다.
4. **최종 보고서 (`Kpop_시청률_분석_최종보고서.docx`)**:

   - 분석 결과와 시각화 자료, 모델 성능 비교를 정리한 공식 보고서입니다.
   - 남녀 시청률이 동반 상승하는 이유에 대한 분석 내용이 포함되어 있습니다.

## 🔍 분석 결과 요약

- **상관관계**: 남성 시청률과 여성 시청률은 아주 강력한 양의 상관관계(약 0.9822)를 보입니다.
- **예측 결과**: 남성 시청률이 **10.0%**일 때,
  - **선형회귀 모델**: 약 **11.32%** 여성 시청률 예상
  - **다항회귀 모델**: 약 **11.78%** 여성 시청률 예상
- **최적 모델**: 단순 직선 모델보다 데이터의 흐름을 유연하게 따라가는 **3차 다항회귀(곡선) 모델**이 더 높은 예측 정확도를 보여주었습니다.

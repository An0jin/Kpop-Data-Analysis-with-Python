# 🌟 에스파 뉴스 데이터 텍스트 마이닝 프로젝트 🌟

안녕하세요! 👋 
이 프로젝트는 **"뉴스에서 가장 많이 언급된 에스파(aespa) 멤버는 누구일까?"** 라는 호기심에서 출발한 재미있는 데이터 분석 프로젝트입니다.

이 레포지토리는 학생들을 위한 **텍스트 마이닝(Text Mining) 교육용 웹/기초 자료**로 만들어졌습니다. 여러분이 평소에 좋아하고 관심 있는 K-Pop 아이돌 데이터를 직접 다뤄보면서, 딱딱하게 느껴졌던 코딩과 인공지능, 그리고 데이터 분석에 대해 쉽고 친근하게 배워볼 수 있습니다! 🚀

> 🤖 **참고(AI 코딩):** 이 프로젝트의 파이썬 데이터 분석 코드 및 시각화 결과물 등은 사람이 직접 하나하나 코딩한 것이 아니라, **인공지능(AI) 어시스턴트를 활용한 AI 코딩(Vibe Coding)** 방식을 통해 쉽고 빠르게 작성되었습니다! 누구나 아이디어만 있다면 AI와 함께 훌륭한 프로젝트를 완성할 수 있습니다.

---

## 📊 데이터 출처
본 분석과 교육에 사용된 뉴스 기사 데이터는 한국언론진흥재단에서 제공하는 뉴스 빅데이터 분석 시스템인 **[빅카인즈(BIG KINDS)](https://www.bigkinds.or.kr/)** 에서 수집 및 활용되었습니다. 양질의 데이터로 정확하고 신뢰성 있는 분석을 체험해 보세요.

## 🎯 무엇을 배울 수 있나요? (교육 목표)
- 친숙한 K-Pop 주제를 통해 데이터 전처리와 텍스트 마이닝의 기초 개념을 재미있게 학습합니다.
- 엄청나게 많은 뉴스 텍스트 속에서 어떤 단어가 가장 자주 등장했는지 통계를 냅니다 (빈도수 분석).
- 분석한 데이터를 파이썬(Python)을 활용하여 파이그래프(원형), 바 차트(막대형), 워드클라우드 등 예쁘고 알기 쉬운 한눈에 들어오는 데이터로 시각화해 봅니다.

## 🏫 교과 연계 안내
이 프로젝트는 다음과 같은 교과목과 연계하여 학습할 수 있습니다.
- **국어**: 뉴스 기사의 언어 사용 양상 분석 및 비판적 미디어 읽기
- **사회**: 뉴스 데이터를 통한 현대 사회의 문화적 현상 및 트렌드 이해

## 📂 주요 파일 안내
- `data.xlsx`: 빅카인즈에서 검색하고 다운로드한 원본 뉴스 데이터입니다.
- `main.ipynb`: 파이썬 코드가 한 줄 한 줄 적혀져 있는 데이터 분석 메인 파일(주피터 노트북)입니다. 코드를 직접 실행해보세요!
- `wordcloud.png`, `bar.png`, `pie.png`: 코드를 직접 실행해서 얻은 예쁜 시각화 결과물입니다.
- `에스파_텍스트마이닝_분석보고서.pdf` / `.docx`: 이 프로젝트를 수행한 최종 결론이 보기 좋게 담긴 분석 보고서입니다.

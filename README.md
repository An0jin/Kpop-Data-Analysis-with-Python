# 🎤 KPOP 데이터 분석 with Python & AI

초·중·고등학생을 대상으로 **파이썬(Python)과 인공지능(AI)**을 활용하여 쉽고 재미있게 데이터를 분석해보는 수업 자료 저장소입니다.
학생들에게 친숙한 **'KPOP'**을 주제로 다양한 데이터를 수집, 분석하고 시각화하며 프로그래밍과 데이터 과학의 기초를 다집니다.

## 🎯 수업 목표

* **흥미 유발:** 내가 좋아하는 아이돌, KPOP 관련 데이터를 직접 다루며 코딩에 대한 장벽을 낮춥니다.
* **파이썬 기초:** 데이터 분석에 필수적인 파이썬 문법과 라이브러리(Pandas, Matplotlib 등)의 활용법을 익힙니다.
* **데이터 리터러시:** 데이터를 수집하고, 의미를 도출하고, 결과를 시각화하며 데이터를 해석하는 능력을 기릅니다.
* **AI 맛보기:** 자연어 처리(NLP), 머신러닝 등을 활용하여 인사이트를 도출해 봅니다.

### 2. 프로젝트 실습 (진행 중)

KPOP 관련 주제로 직접 데이터를 수집하고 분석하는 프로젝트를 진행합니다.

* ✨ **프로젝트 1: 뉴스에서 많이 언급된 에스파 멤버는?**
  * **핵심 분석:** 텍스트 마이닝 (Text Mining)
  * **연계 과목:** **국어** (언어의 사용 양상과 의미 분석), **사회** (대중 매체의 영향력과 사회·문화적 현상 이해)
  * **내용:** 뉴스 기사의 텍스트 데이터를 수집해 언어를 분석함으로써 대중 매체에서 가장 많이 언급되는 에스파(aespa) 멤버가 누구인지 알아봅니다.
  * **주요 기술:** 형태소 분석, 워드클라우드(WordCloud) 시각화
  * **폴더:** [`/뉴스에서 많이 언급된 에스파 멤버는?`](./뉴스에서 많이 언급된 에스파 멤버는?/)

* 📈 **프로젝트 2: 남성 시청률이 10%면 여성 시청률을 몇%일까?**
  * **핵심 분석:** 회귀 분석 (Regression Analysis)
  * **연계 과목:** **사회** (성별 시청 행태와 대중 문화 소비 트렌드 분석)
  * **내용:** K-Pop 프로그램의 남성 시청률을 통해 여성 시청률을 예측하는 모델을 만들어보고, 데이터 사이언스의 기초를 학습합니다.
  * **주요 기술:** 선형회귀 및 다항회귀 모델링, 모델 성능 평가
  * **폴더:** [`/남성시청률이 10%면 여성 시청률을 몇%일까?`](./남성시청률이 10%면 여성 시청률을 몇%일까?/)
